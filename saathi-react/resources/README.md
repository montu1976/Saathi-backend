# App store icons

Place these files here, then run from `saathi-react`:

```powershell
npm run assets:generate
npx cap sync
```

| File | Size | Notes |
|------|------|--------|
| `icon.png` | 1024×1024 | Required. Square, warm Saathi logo, minimal padding |
| `splash.png` | 2732×2732 | Optional. Logo centered on `#e07a5f` background |

Export from Figma/Canva or upscale `public/saathi-logo.svg` to PNG.
